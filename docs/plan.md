# Projeto de Landing Page para App Fitness

Pagina com fundo branco e fonte Roboto como principal.
Objetivo é ter uma landing page clean, branca e sofisticada

## Seçâo HEADER:

Logo no canto esquerdo
- A logo é um F em um circulo (feito com CSS)
- O fundo do circulo é VERDE ESCURO

Botão de Download no canto direito
- fundo verde, cor branca
- borda arredondada

## Seçâo HERO :

A hero deve ter altura 90vh, divido em 3 areas horizontais

Esquerda:
Apenas a Headline, sem ser grande de mais, sem destaque (o destaque é o print do app no meio)
"O App Perfeito para acompanhar sua vida Fitness."

Centro:
Em destaque a imagem images/print_1.png que é um print do app.

Direita:
Uma lista de pontos-chave sobre o app
- ❤️​ Saúde
- 🏃‍♂️‍➡️ Exercicio
- 🤝​ Amigos

## Seçâo de FUNCIONALIDADES:

Vão ser usadas as imagens images/print_2.png, images/print_3.png e images/prin_4.png que serão exibidas da seguinte forma:
- print / texto 
- texto / print
- print / texto

Ou seja, ira alternar entre o print na esquerda e um texto explicativo na direita, e vice e versa

Descrição breve de cada print para você gera um texto adequado

- print_2.png = Contagem de Calorias
- print_3.png = Exercicios Específicos (Corrida, Musculação e etc)
- print_4.png = Compartilhamento com os amigos

##Seçâo de CTA (CAll-TO-ACTION):
Botão grande de Download com um texto abaixo dele descrevendo
"Totalmente Gratuito"

RODAPE:
Apenas com o texto "Todos os direitos reservados."
